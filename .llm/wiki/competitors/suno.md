# Suno

> Last updated: 2026-04-05

## Key Features

### AI Music Generation
- Text-to-music with genre/mood prompts
- V4 model: 2-minute generations, higher quality
- Custom mode: separate lyrics + style prompt
- Extend: continue a generation from any point
- Cover: re-sing with different voice

### UX Patterns
- Very simple: prompt → generate → listen → share
- No DAW-like editing capabilities
- Library-based management (playlists)
- Social features (share, remix)

## Differentiation from ACE-Step
- Suno is consumer-focused, ACE-Step is creator-focused
- ACE-Step offers per-track editing, mixing, effects
- ACE-Step supports multi-track orchestration
- Suno has better voice synthesis currently
