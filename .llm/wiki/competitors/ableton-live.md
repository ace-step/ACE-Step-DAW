# Ableton Live 12

> Last updated: 2026-04-05

## Key Features (interaction-detail level)

### Mixing
- Mixer panel: vertical channel strips, foldable groups, send knobs per track
- Group tracks: nestable, shows sub-clip overview when folded
- Cmd+Click for multi-select, color applies to all sub-tracks
- Source: [Ableton Mixing Docs](https://www.ableton.com/en/live-manual/12/mixing/)

### MIDI Editing
- Velocity editor below piano roll, adjustable height
- Multi-clip editing: select clips across tracks, edit simultaneously
- Note stretch: Shift+drag note end
- Source: [Ableton MIDI Docs](https://www.ableton.com/en/live-manual/12/editing-midi/)

### Automation
- Breakpoint envelopes on arrangement clips
- Session view: per-clip envelopes
- Automation arm per parameter
- Source: [Ableton Automation Docs](https://www.ableton.com/en/live-manual/12/automation-and-editing-envelopes/)

### Session View
- Follow actions with probability (A/B split)
- Quantized clip launching
- Arrangement recording from session
- Source: [Ableton Recording Docs](https://www.ableton.com/en/live-manual/12/recording-new-clips/)

## Relevance to ACE-Step
- Session View follow actions implemented in our SessionState type
