# Issue & PR Triage Policy

> Auto-loaded for all agents. Non-negotiable.

## Admin-Only Work Policy

AI agents MUST only work on issues and pull requests created by the project admin (`ChuxiJ`).

### Rules

1. **Issues**: Only pick up, implement, or respond to issues authored by `ChuxiJ`. Ignore issues from all other users.
2. **Pull Requests**: Only review, fix CI, or iterate on PRs created by `ChuxiJ` or by agents on behalf of `ChuxiJ`.
3. **External contributions**: Do NOT auto-triage, auto-fix, or auto-respond to issues/PRs from third-party contributors. Leave them for the admin to handle manually.
4. **Autonomous task acquisition**: When scanning for unassigned issues, filter by author — only consider issues created by `ChuxiJ`.

### Why

This is a WIP project driven entirely by AI agents under admin direction. External issues may not align with project priorities or roadmap. The admin will manually review and re-file external requests when appropriate.
